Gaming Room 2.0 - striking visual identity & responsive web for an epic contest

x-kom launched the second edition of its Gaming Room contest, offering a dream makeover of a viewer's room. Our mission was to refresh the contest's visual identity and build a responsive site that handles submissions, public voting, and features Alienware product promotions.

Goals

The primary goal of this project was to design and develop a modern, responsive website for the second edition of the Gaming Room contest. The platform needed to handle all key user interactions - from submitting entries, through public voting, to displaying live rankings. In addition, it was important to refresh the visual layer of the contest so that it better resonated with the gaming audience, and to integrate Alienware promotions into the user journey in a way that felt engaging but not intrusive.

Design Process

Audit & StrategyVisual IdentityWebsite DesignPromotion Integration

Final Design (UI + Responsive)

The end result is a bold, gaming-inspired web platform that flawlessly works across devices. It delivers a seamless UX-users can submit designs, vote, and explore Alienware offers, all in a cohesive visual universe.

Final Design

  • Higher user participation compared to the first edition
  • Intuitive user engagement through voting/submissions
  • Smooth integration of Alienware products into the contest ecosystem
